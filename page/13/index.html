JS对象基本用法

Posted on 2019-09-20 Views: Valine:

声明对象的两种语法

  1. let obj = {‘name’ : ‘frank’,’age’:21 } 常用，写起来简单
  2. let obj = new Object({‘name’:’frank’})

如何删除对象的属性

  1. delete obj.xxx
  2. delete obj[‘xxx’]

如何查看对象的属性

  1. 查看自身所有属性Object.keys(obj)
  2. 查看自身所有值Object.values
  3. 查看自身所有属性和值obj.values和obj.entries
  4. 查看自身+共有属性,console.dir(obj)
  5. 判断一个属性是否是自身的，如果是返回true,obj.hasOwnProperty(‘toString’)

如何修改或增加对象的属性

  1. 直接赋值
    1. let obj = {name:’frank’} // name 是字符串
    2. obj.name = ‘frank’ // name是字符串
    3. obj[‘name’] = ‘frank’
  2. 批量赋值 Object.assign(obj,{age:18,gender:’main’})

‘name’ in obj和obj.hasOwnProperty(‘name’) 的区别

  1. ‘name’ in obj 是’name’是否包含在obj的全部属性中，也就是说如果’name’在__proto__中也会返回true
  2. obj.hasOwnProperty(‘name’) 指的是’name’是否包含在obj的本身属性中，不会去检查obj的__proto__中是否有’name’

JavaScript的诞生

Posted on 2019-09-12 Views: Valine:

JavaScript的发明者

  • 网景公司的员工Brendan Eich,用10天时间设计出了Javascript。

JavaScript的设计思路

  • 借鉴C语言的基本语法；
  • 借鉴Java语言的数据类型和内存管理；
  • 借鉴Scheme语言，将函数提升到”第一等公民”（first class）的地位；
  • 借鉴Self语言，使用基于原型（prototype）的继承机制。

完整的JavaScript包括以下几个部分

  • ECMAScript，描述了该语言的语法和基本对象
  • 文档对象模型（DOM），描述处理网页内容的方法和接口
  • 浏览器对象模型（BOM），描述与浏览器进行交互的方法和接口

JavaScript的基本特点

  • 是一种解释性脚本语言（代码不进行预编译）。
  • 主要用来向HTML页面添加交互行为。
  • 可以直接嵌入HTML页面，但写成单独的js文件有利于结构和行为的分离。

JavaScript常用完成的任务

  • 嵌入动态文本于HTML页面
  • 对浏览器事件作出响应
  • 读写HTML元素
  • 在数据被提交到服务器之前验证数据
  • 检测访客的浏览器信息
  • 控制cookies，包括创建和修改等

CSS知识总结

Posted on 2019-09-10 Views: Valine:

浏览器渲染过程

  1. 根据HTML构建HTML树
  2. 根据CSS构建CSS树
  3. 将前面两棵树合并成一棵渲染树
  4. Layout布局，这步相当于是素描，摆放文档流的位置，计算大小
  5. Paint绘制，这部分相当于是天色，把文字，阴影等都画出来
  6. Compose合成（根据层叠关系展示画面）

三种更新方式

  1. 全部走一遍
  2. 跳过layout,直接repaint + composite
  3. 跳过layout和paint,直接composite

如何查看渲染过程

  1. 调出console
  2. 按esc，选左边三个点
  3. 选择rendering

查看css各属性触发什么？

  1. CSSrigger.com

CSS动画优化

  1. 不要用left之类的属性,用transform做动画
  2. 使用will-change或translate

CSS 动画的两种做法

  1. transition。在某个需要动画的标签里写，transition: all 1s;。在此标签的伪类比如hover里面写:transform: scale(1.3,1.3);
  2. animation。先要定义@keyframes，然后在需要动画的标签里面写属性如:animation: .5s wrapper infinite alternate-reverse

浅析URL

Posted on 2019-08-29 Views: Valine:

URL 包含哪几部分，每部分分别有什么作用

  • URL:包含协议+域名或IP+端口号+路径+查询字符串+锚点。
  • 协议:说明用什么协议
  • 域名:代表一台机器
  • 路径:使用不同路径请求不同页面。
  • 查询字符串:通过查询字符串，实现同一个页面，不同的内容。
  • 锚点:锚点定位查询结果的第几个，或者是某一块内容

DNS 的作用是什么，nslookup 命令怎么用

  • DNS:把域名和IP联系起来的系统就称为DNS。当你在浏览器中输入一个网址，浏览器首先会向DNS服务商（比如电信）询问此网址对应什么IP,之后电信会回答一个ip。
  • nslookup: +一个域名，返回一个ip地址

IP 的作用是什么，ping 命令怎么用

  • IP的作用:一.如何定位一台设备。二.如何封装数据报文，以跟其他设备交流。

域名是什么，分别哪几类域名

  • com是顶级域名，xxx.com是二级域名，俗称一级域名。www.xxx.com是三级域名（俗称二级域名），他们是父子关系。有可能是同一家，也有可能不是。
