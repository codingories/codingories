Posted on 2020-05-05 Views: Valine:

一. Navicat 的使用

  • 连接，连接名可以随便填写，主机名需要根据需要填写

  • 右键顶部可以新建数据库，字符集尽量选择 utf-8,排序规则选择 utf-8general-ci

  • 在表上右键可以新建表，也可以通过查询输入 sql 命令实现

  • 新建列，然后保存表名

  • 进一步设计表，设计表结构

  • 解决没有主键的问题。

  • 数据的复制，语句查询，表转存，导入 sql 文件，

  • mac 下工具因为是 essential 版，不是 premium 版 没有数据传输功能，这个功能非常方便把线下的库放到线上

    二. pycharm 简单使用

  • pure python 选择解释器，右键 add local,找到 python 可执行文件的路径,make available to all projects

  • 如果是虚拟环境要选择虚拟环境的目录下的 script 中的 python,因为我们选择了 virtualenvwarpper 所以选择 existing interpret 就可以了

  • pycharm 调试代码 打断点,以 debug 的方式运行，按 f6 单步调试，直接运行到另外一个断点处，f8,

  • f5 的作用是 进入到某个函数去调用

  • pycharm 快捷键，
