  • React.StrictMode 标签的作用用来检查代码

  • tsconfig 是配置文件
  • 总结 index.html，app.tsx,index.tsc,tsconfig 一定要知道，稍微知道 react-app-env.d.ts 是声明文件就可以了

三. react css 相关配置

  1. css normalize 的配置,作用是让不同浏览器上的样式尽量相近，举例：让不同的浏览器的按钮尽量相同,只要在 index.css 中加入@import-normalize 即可
  2. css normalize 和 css reset 的区别是 css normalize 要把默认样式基本一样，css reset 是把默认样式去掉

四. 让 react 项目支持 scss

  • 最难的技术问题，1.我想再某个项目中支持 scss。2. 需要用 node-sass,它有两个缺点: 下载速度慢，本地编译慢。3.于是我想改用 dart-sass 代替 node-scss。4.但是 React 只吃吃 node-sass 不支持 dart-sass。5.经过我的努力搜索和碳九。6.我发现 npm6.9 支持一个新功能，叫 package alias。7.通过 npm install node-sass@npm:dart-sass 即可偷梁换柱瞒天过海。8.最后达成了我的目标。
