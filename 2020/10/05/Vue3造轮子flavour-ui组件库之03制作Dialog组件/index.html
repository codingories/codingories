8. 让 Dialog 支持自定义内容

  • 使用插槽 slot,实现 title 和内容都是自定义的
  • 使用具名插槽，解决 title 也可以加样式的问题
  • DialogDemo.vue
  • Dialog.vue

  • 总结:具名插槽的用法

    • 首先在外面写上 v-slot:名字,里面写上 slot name=

9. 使用新组件：Teleport，把 Dialog 移到 body 下

  • 防止 Dialog 被遮挡，遮挡原因是因为 z-index 层叠上下文的关系，z-index:1 里面的 z-index:10,会被 z-index:2 盖掉，也就是说比别人大还是小要看所在的环境,举例:清华里面的最后一名比三本的第一名要大
  • 解决办法就是把 Dialog 移到外面,直接出现在 body 下面
  • 简单理解 Teleport 就是传送门，把 Dialog 传送到 Body 下面

10. 一句话打开 Dialog

  • 场景:不想声明 visible 变量，然后改变它的值
  • 技术点：动态挂载组件

  • lib/openDialog.ts
