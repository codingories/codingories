9. Vue3 编程模型

  • 内部数据 VS 父子数据

  • 内部数据的意思是，所有的数据都是在组件内部，好处是写起来方便，坏处是没办法和外面沟通

  • 第二个写的 switch，接口 value 和 @input 事件,父子数据更常用

10. v-model 对 父子之间的数据交流进行简化

  • 要求

    • 属性名任意，假设为 x
    • 事件名必须为 “update:x”

  • 效果

    • <Switch :value="y" @update:value="y = \$event"/>
    • 可以简写为<Switch v-model:value="y" />

  • SwitchDemo.vue

  • Switch.vue

11. 知识点总结

  • value=”true” 和 :value=”true” 的区别
  • 使用 CSS transition 添加过渡动画
  • 使用 ref 创建内部数据
  • 使用 :value 和 @input 让父子组件进行交流(组件通信)
  • 使用 $event
  • 使用 v-model

12. Vue 2 和 Vue 3 的区别

  • 新 v-model 代替以前的 v-model 和 .sync
  • 新增 context.emit，与 this.$emit 作用相同
    • methods 里可以用 this.$emit,但是以后不会用 methods，setup 里面写一个函数 return 出来就是 methods,Vue3 基本就用 context.emit
